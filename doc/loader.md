## bsd bootloader infos


### help

 shows help
 ```
 ?
 ```


### disk infos

list known disks
```
lsdev
```

select disk
```
set currdev="diskXsY"
```

set mountpoint
```
set vfs.root.mountfrom="ufs:/dev/adaXsYa"
```

### variables

show bootloader variables
```
show
```

### boot

boot selected disk
```
boot
```
