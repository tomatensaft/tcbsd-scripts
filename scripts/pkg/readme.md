## package management


### setup local repo

get tcbsd repo
``` 
setup_local_repo.sh --get
``` 

### setup portsnap system

script for setup portsnap
``` 
setup_portsnap.sh
``` 

### switch repo system

switch to beckhoff
``` 
switch_pkg_repo.sh --beckhoff
``` 
switch to freebsd
``` 
switch_pkg_repo.sh --official
```
